# 🚦 CivicFlow AI — Incident Intelligence System

An AI-powered full-stack application designed to analyze transit disruptions and provide actionable insights including severity classification, root cause analysis, and operational recommendations.

---

## 📌 Overview

CivicFlow AI helps transit authorities and operators:
- Detect and analyze incidents in real-time
- Generate AI-powered insights using LLMs
- Improve passenger communication
- Support faster operational decision-making

---

## ⚙️ Tech Stack

### Backend
- **FastAPI (Python)**
- **Oracle SQL (Local Database)**
- AI Integration (Gemini API / LLM)
- REST API architecture

### Frontend
- **React + Vite**
- TypeScript
- Modern UI components
- Dashboard & Analytics panels

### Dev Tools
- Git + GitHub
- ngrok (for public API exposure)
- VS Code

---

## 🧠 Key Features

### 🔍 Incident Analysis
- Severity classification (Low / Medium / High)
- Root cause detection
- AI-generated recommendations
- Confidence scoring

### 📊 Dashboard
- Total incidents tracking
- AI usage metrics
- Remaining API quota
- Interactive charts (severity distribution)

### 🧾 AI Insights Panel
- Passenger communication message
- Executive summary
- Actionable recommendations

### ⚡ Performance Optimization
- API response caching
- Rate limiting for AI usage
- Fallback logic (cache vs live AI)
